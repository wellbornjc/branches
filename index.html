Branches is a tool for creating branching narratives for interactive fiction.

- Start by naming your project at the top of the page by clicking the litte pencil icon below Project Name...

- If you wish to return to the top of the page click the Return To Title button at the top of the page.

- To add a branch click the Create a New Branch button at the top left of the page.

- Once a branch is created it will appear at the bottom of the page. Click and drag the top tab of the branch to place it where you need on the page.

- You can edit the information in the branch by clicking the Edit button or delete it entirely by clicking the Delete button.

- To draw a line to another branch click and drag from the Connect button and release it on another branch you wish to connect.

- You can delete connected lines by clicking on them.

- In Options you can delete all branches in the project.

- To see this information again click the Help button in Options.
